# Nlw-Heat
Projeto desenvolvido no evento Nlw-Heat
## Rodar Projeto


```bash
#Clonar repositório
$ git clone https://github.com/OtavioEscalnte/Nlw-Heat.git
$ cd Nlw-Heat

```
## Rodar API
```bash
$ cd node_heat
#instalar dependências
$ yarn install
#Rodar api
$ yarn dev

```
## Rodar Projeto Web

```bash
$ cd web
#instalar dependências
$ cd yarn install 
#rodar aplicação 
$ yarn dev
```
## Rodar Projeto Mobile
```bash
$ cd nlwheatapp
#instalar dependências
$ cd yarn install
#rodar aplicativo
$ expo start
```
